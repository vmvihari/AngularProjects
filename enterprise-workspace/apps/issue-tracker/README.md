# IssueTracker (Angular)

This is the primary frontend Angular application for the Enterprise Issue Tracker bootcamp curriculum.

## Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- Angular CLI

## Running the Application
1. Navigate to the root of the Angular workspace in your terminal:
   ```bash
   cd enterprise-workspace
   ```
2. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```
3. Start the Angular development server:
   ```bash
   npm start
   ```

The application will compile and become available at `http://localhost:4200`.

## Backend API
This application relies on the `.NET` API to fetch real data. Ensure that you have the `IssueTracker.Api` running on `http://localhost:5000` alongside this Angular application.
