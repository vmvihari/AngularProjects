# Enterprise Angular for .NET Developers

Welcome to the **Enterprise Angular for .NET Developers** course repository. This repository contains the documentation and source code for building an Enterprise Issue Tracking System using modern Angular (v18+).

## Course Overview
The purpose of this curriculum is to leverage your senior .NET background to rapidly build expert-level proficiency in Angular. We will skip the absolute basics and focus directly on modern, enterprise-grade architecture so you can confidently architect applications and market yourself for senior Angular roles.
