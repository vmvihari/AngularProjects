# Phase 9, Lesson 1: Enterprise Reactive Forms

Welcome to Phase 9: Forms & Validation!

We are going to use Angular's **Reactive Forms**, which is the industry standard for building robust, scalable forms in enterprise Angular applications. In Reactive Forms, you build the form model directly in your TypeScript component using `FormGroup` and `FormControl`, giving you ultimate synchronous control over validation state!

## Your Task: Build the Create Issue Feature

### 1. Generate the Library
As always, we want to encapsulate our new feature in its own library so it can be lazy-loaded! Run this command:

```bash
npx nx g @nx/angular:library --name=feature-issue-create --directory=libs/issues/feature-issue-create --standalone
```

### 2. Update the Global IssueStore
Before we build the UI, our global `IssueStore` needs to know how to save a new issue!
Open `libs/issues/data-access/src/lib/issue.store.ts`.

First, let's update our `Issue` interface to support a description:
```typescript
export interface Issue {
  id: number;
  title: string;
  status: string;
  description?: string; // <-- Add this!
}
```

Next, add an `addIssue` method inside `withMethods`:

```typescript
    // Inside withMethods...
    addIssue(title: string, description: string) {
      // 1. Optimistic UI Update (Immediate!)
      const tempId = Date.now(); // Temp ID until server responds
      patchState(store, (state) => ({
        issues: [...state.issues, { id: tempId, title, description, status: 'Open' }]
      }));

      // 2. Background Sync
      http.post(`${environment.apiUrl}/issues`, { title, description, status: 'Open' }).subscribe();
    },
```

### 3. Build the Form Component
Open the default component generated by Nx: `libs/issues/feature-issue-create/src/lib/feature-issue-create/feature-issue-create.ts`.

Copy in the code below. Notice how we import `ReactiveFormsModule`, inject our `IssueStore`, and strictly type our form!

```typescript
import { Component, inject } from '@angular/core';
import { Router } from '@angular/router';
import { ReactiveFormsModule, FormGroup, FormControl, Validators } from '@angular/forms';
import { IssueStore } from '@enterprise-workspace/data-access';
import { NgClass } from '@angular/common'; // We will use this for CSS later

@Component({
  selector: 'lib-feature-issue-create',
  standalone: true,
  imports: [ReactiveFormsModule, NgClass],
  templateUrl: './feature-issue-create.html',
  styleUrl: './feature-issue-create.css'
})
export class FeatureIssueCreate {
  public issueStore = inject(IssueStore);
  private router = inject(Router);

  // 1. Create the STRICT form model!
  public issueForm = new FormGroup({
    title: new FormControl<string>('', { 
      nonNullable: true, 
      validators: [Validators.required, Validators.minLength(5)] 
    }),
    description: new FormControl<string>('', { 
      nonNullable: true 
    })
  });

  // 2. Define the submit action
  onSubmit() {
    if (this.issueForm.valid) {
      const { title, description } = this.issueForm.getRawValue();
      
      // Save it using our global store!
      this.issueStore.addIssue(title, description);
      
      // Navigate back to the Issue Manager!
      this.router.navigate(['/issues']); 
    }
  }
}
```

### 4. Build the Template
Open the template file: `libs/issues/feature-issue-create/src/lib/feature-issue-create/feature-issue-create.html` and replace it entirely with:

```html
<div class="form-container">
  <h2>Create New Issue</h2>
  
  <!-- Bind the [formGroup] directive to our TS model! -->
  <form [formGroup]="issueForm" (ngSubmit)="onSubmit()">
    
    <div class="form-group">
      <label>Issue Title</label>
      <!-- Bind this input to the specific FormControl -->
      <input type="text" formControlName="title" placeholder="E.g., Server is down..." />
      
      <!-- Validation Error Rendering -->
      @if (issueForm.controls.title.touched && issueForm.controls.title.errors?.['required']) {
        <span class="error-msg">Title is required.</span>
      }
      @if (issueForm.controls.title.touched && issueForm.controls.title.errors?.['minlength']) {
        <span class="error-msg">Title must be at least 5 characters long.</span>
      }
    </div>

    <div class="form-group">
      <label>Description</label>
      <textarea formControlName="description" rows="4"></textarea>
    </div>

    <!-- Disable the submit button natively if the form is completely invalid! -->
    <button type="submit" class="submit-btn" [disabled]="issueForm.invalid">
      Create Issue
    </button>
  </form>
</div>
```
